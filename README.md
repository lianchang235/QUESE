Mini-ROV 地面站控制器 README

基于STM32G030C8T6的水下机器人手持控制终端
 
 项目简介
 
本项目为嵌入式综合考核Mini-ROV水下机器人地面站控制器的完整实现。通过双轴摇杆采集用户控制指令，经STM32处理后按照标准通信协议打包发送给水下主控板，同时接收并解析水下机器人的实时状态数据，在OLED屏幕上进行双页面切换显示。
 
核心功能详解
 
1. 摇杆采集与控制映射
 
- 采用单通道轮询模式读取ADC值，采样时间1.5个时钟周期
- 摇杆中心值固定为2048，输出范围线性映射为-5.0N ~ +5.0N
- 左摇杆Y轴：控制ROV前后推力
- 左摇杆X轴：控制ROV左右转向扭矩
- 右摇杆Y轴：控制ROV上下推力
 
2. 通信系统
 
- 采用异步串口通信，波特率115200，8位数据位，1位停止位，无校验
- 下行控制帧发送频率：10Hz（每100ms发送一次）
- 上行状态帧采用中断方式接收，通过状态机解析，完美解决串口粘包问题
- 支持自动发送状态请求帧，主动获取水下机器人数据
 
3. 按键与显示系统
 
- 两个摇杆按键均带20ms硬件消抖
- 右摇杆按键：切换机械臂角度
- 左摇杆按键：切换OLED显示页面
- OLED显示分为两个页面：- 页面0：显示Y推力、Z推力、Yaw扭矩
- 页面1：显示四个电机的单独推力、机械臂角度
- 顶部状态栏固定显示系统运行时间和总功率
 
4. 数据处理
 
- 实现四电机推力分配算法，将三维控制量转换为四个推进器的独立推力
- 自动计算水下机器人总功耗（12V供电）
- 完整的帧头、帧尾和校验和验证，确保通信可靠性

   通信协议规范
 
1. 下行控制帧（地面站 → ROV，23字节）
 
总长度23字节，偏移范围0~22
 
- 偏移0，长度2字节：帧头，固定为0xFA 0xAF
- 偏移2，长度1字节：帧类型，固定为0x49('I')
- 偏移3，长度4字节：Y轴推力，float类型，范围-10.0N ~ +10.0N
- 偏移7，长度4字节：X轴推力，float类型，预留，固定为0.0N
- 偏移11，长度4字节：Z轴推力，float类型，范围-10.0N ~ +10.0N
- 偏移15，长度4字节：Yaw扭矩，float类型，范围-10.0N·m ~ +10.0N·m
- 偏移19，长度1字节：机械臂角度，0x00=0°, 0xFF=45°
- 偏移20，长度1字节：校验和，字节0~19的异或值
- 偏移21，长度1字节：帧尾1，固定为0xFB
- 偏移22，长度1字节：帧尾2，固定为0xBF
 
2. 上行状态帧（ROV → 地面站，15字节）
 
总长度15字节，偏移范围0~14
 
- 偏移0，长度2字节：帧头，固定为0xFA 0xAF
- 偏移2，长度1字节：帧类型，固定为0x53('S')
- 偏移3，长度4字节：水温，float类型，单位℃
- 偏移7，长度4字节：工作电流，float类型，单位A
- 偏移11，长度1字节：漏水报警，0x00=正常，0x01=漏水
- 偏移12，长度1字节：预留，固定为0x00
- 偏移13，长度1字节：校验和，字节0~12的异或值
- 偏移14，长度1字节：帧尾1，固定为0xFB
- 偏移15，长度1字节：帧尾2，固定为0xBF
 
3. 状态请求帧（地面站 → ROV，5字节）
 
总长度5字节，偏移范围0~4
 
- 偏移0，长度2字节：帧头，固定为0xFA 0xAF
- 偏移2，长度1字节：帧类型，固定为0x52('R')
- 偏移3，长度1字节：帧尾1，固定为0xFB
- 偏移4，长度1字节：帧尾2，固定为0xBF
  代码结构
全局变量
uint8_t oled_page;          // OLED当前页面
uint8_t servo_angle;        // 机械臂角度
float total_power;          // 总功率
float rov_current;          // 工作电流
float rov_temperature;      // 水温
uint8_t water_alert;        // 漏水报警
uint8_t uart_rx_buf[15];    // 串口接收缓冲区
uint8_t uart_rx_ptr;        // 接收指针
uint8_t uart_rx_byte;       // 单字节接收变量
 
 
核心数据结构
 
-  DownDataFrame ：下行控制帧联合体，支持字节数组和结构体两种访问方式
-  UpDataFrame ：上行状态帧联合体，支持字节数组和结构体两种访问方式
- 所有结构体均使用 #pragma pack(1) 强制1字节对齐，避免内存对齐问题
 
核心函数
 
-  UART_Send_Data() ：打包并发送下行控制帧
-  UART_Send_Request() ：发送状态请求帧
-  ADC_Read() ：读取指定ADC通道的值
-  Key_Scan() ：按键扫描与消抖处理
-  Calculate_Motor_Thrust() ：四电机推力分配计算
-  Parse_Up_Frame() ：解析上行状态帧
-  HAL_UART_RxCpltCallback() ：串口接收中断回调函数
